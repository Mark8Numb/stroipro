=== ОПИСАНИЕ КЛЮЧЕВЫХ ФИЧ НА ДАТУ ===

1. Офлайн PWA-приложение для учёта строительных работ
   — Полностью работает без интернета
   — Устанавливается на телефон через браузер

2. Журнал работ
   — Добавление записей с датой, видом работ, объёмом
   — Редактирование, удаление, скрытие записей
   — Сортировка по дате, объёму, сумме

3. Склад материалов
   — 12+ основных позиций + расходники
   — Приход, расход, остаток в штуках, пачках, метрах
   — История приходов с датой и временем
   — Цветовая индикация дефицита (красный — критично, жёлтый — заканчивается)

4. Задачи на день
   — Добавление задач с датой и временем
   — Автоматический перенос просроченных задач на сегодня
   — Отметка выполненных → скрытие через 3 секунды

5. Заработок и аналитика
   — Расчёт заработка за день, неделю, месяц
   — Учёт налога 13%
   — График динамики выработки
   — Разбивка по корпусам и видам работ

6. AI-анализ (локальный, без сервера)
   — Оценка продуктивности (1–10)
   — Прогноз сдачи объекта
   — Сравнение работников по выработке

7. Работники и бригады
   — Добавление работников с привязкой к бригаде
   — Распределение объёма и заработка между несколькими работниками

8. Отчёты для заказчика
   — Генерация отчёта по объекту (сумма, объём, остатки, прогноз)
   — Отправка в WhatsApp / Telegram

9. Монетизация
   — Бесплатная версия: до 50 записей
   — PRO-подписка (499 ₽/мес, в будущем 1 490 ₽/мес)

10. Технические характеристики
    — Размер: 104–137 КБ
    — Зависимости: 0 внешних библиотек
    — Хранение: localStorage
    — Код: Vanilla JS, CSS, HTML
