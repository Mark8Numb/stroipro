📖 Инструкция: как работать с приложением

🚀 Как начать пользоваться?

1. Откройте приложение → нажмите «+ Добавить объект».
2. Введите название объекта (например, «ЖК Северный парк»).
3. Внутри объекта нажмите «+ Корпус» и укажите название.
4. Теперь можно добавлять виды работ и материалы.

🏗️ Как добавить вид работ и материалы?

Вид работ: в карточке корпуса → «Виды работ» → «+» → укажите название и расценку (₽ за м²/шт).
Материалы: раздел «СКЛАД» → «+ Материал» → название, единица измерения, количество, цена.

🗑️ Как удалить всё (сбросить данные)?

В приложении предусмотрена кнопка «Сбросить все данные» в разделе «ПРОЧЕЕ» → «Настройки». Нажмите её, чтобы полностью очистить объекты, корпуса и материалы.
⚠️ Действие необратимо (данные хранятся только локально).

➕ Объект → Корпус → Работы

Структура простая: Объект (например, ТРЦ) → Корпус (секция А) → Виды работ (монтаж каркаса). Для каждого вида работ можно задать объём и отслеживать прогресс.

💡 Все данные сохраняются автоматически в вашем браузере. Приложение работает без интернета после первого открытия.
